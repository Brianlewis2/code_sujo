CODE SUJO

No dia 23 de março de 2025 nas profundezas do código e da conexão entre irmãos de alma nasceu o grupo CODE SUJO Muito mais do que um grupo de programação essa família surgiu da união de jovens mentes que não aceitavam os limites impostos pelo sistema

Cansados de ver códigos limpinhos que escondem a verdadeira essência do programador raiz os membros decidiram criar seu próprio caminho Cada linha escrita carrega suor luta e paixão Aqui o código é sujo porque é real porque é feito na guerra na madrugada na força da ideia

CODE SUJO representa a rebeldia digital a criatividade sem filtro a irmandade entre os que vivem da lógica mas com coração A cada projeto criado a cada sistema quebrado e reconstruído a essência do grupo se fortalece Família sempre código sempre
